# 🚀 Getting Started

## Clone the repo
```bash
git clone https://github.com/RoBoCRAFTYT01/data-structure.git
cd data-structure
```

## Install dependencies (choose one)
```bash
# Using bun
bun install

# Using npm
npm install

# Using pnpm
pnpm install
```

## Build
```bash
# Using bun
bun run build

# Using npm
npm run build

# Using pnpm
pnpm run build
```

## Run tests
```bash
bun test
# or
npm test
# or
pnpm test
```
