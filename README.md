# 🧪 Tests Branch

This branch (`feature/tests`) contains **unit tests and benchmarks** for the Data Structure project.  
It is dedicated only to testing code — no core source files are maintained here.

---

## 🚀 Running Tests

### Using Bun
```bash
bun run test
```

### Using Npm
```bash
npm run test
```

## 🛠️ Tools & Frameworks
- Typescript
- Jest
- Bun (optional runner)

## 📂 Structure

```bash
tests/

```
